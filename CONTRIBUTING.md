# Contributing to SheetDiff

Thank you for your interest in contributing.

## Development setup

1. Clone the repository.
2. Install Node 22+, pnpm 9+, and Python 3.12+.
3. Run `pnpm install` at the repo root.
4. Start the worker and web apps (see README).

## Pull requests

- Keep changes focused on a single concern.
- Ensure `pnpm` build/typecheck and worker `pytest` pass.
- Use conventional commit messages.
